このツールについて

FANY（ticket.fany.lol）の購入・申込履歴を、公演名・会場・公演日・ステータスで自由に絞り込めるビューワーです。 ブックマークレットを使って FANY の履歴ページからデータを収集し、このビューワーに表示します。

🔒 プライバシーについて
  • FANY のログイン情報（ID・パスワード）はこのツールに送信されません
  • 取得した履歴データはあなたの端末の LocalStorage にのみ保存されます
  • 外部サーバーへのデータ送信は一切ありません

ブックマークレットの設定

Safari ではリンクのドラッグができないため、手動でブックマークを編集します。

  1. 適当なページをブックマークに追加
    ticket.fany.lol/history など、どのページでも構いません。 共有ボタン（↑）から「ブックマークを追加」します。
  2. ブックマークを編集して URL を書き換える
    ブックマーク一覧を開いて、いま追加したブックマークを長押し →「編集」を選択。 名前を「FANY履歴取得」に変更し、URL 欄を全選択して以下のコードに置き換えてください。

使い方

1 FANY にログインした状態で履歴ページを開く

https://ticket.fany.lol/history を開いてください。ログインが必要な場合は先にログインしてください。

2 ブックマークレット「FANY履歴取得」を実行

Safari: アドレスバーに「FANY」と入力してブックマーク候補から選ぶか、ブックマーク一覧から「FANY履歴取得」をタップします。
Chrome: ブックマークバーの「FANY履歴取得」をクリックします。

3 自動でビューワーが開き、データが転送される

ブラウザのタイトルに「取得中 N/19...」と表示されながら全ページを自動取得します。 取得完了後にビューワータブへデータが転送され、一覧が表示されます。

データはあなたの端末に保存されます。次回ビューワーを開いたときも前回のデータが表示されます。 最新の履歴を反映するには再度ブックマークレットを実行してください。

トラブルシューティング

「セッション切れ」と表示される

FANY に再ログインしてから実行してください。

Safari でブックマークレットが動かない

URL 欄が「javascript:」から始まっているか確認してください。 「javascript:」の部分が消えてしまっている場合は、テキストを再度コピーして貼り直してください。

ビューワーが開いたがデータが表示されない

取得完了後にビューワーが開いていなかった可能性があります。 ブックマークレットを再実行するか、ビューワーを開いた状態でもう一度実行してみてください。
